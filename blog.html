1. What is the Document Object Model (DOM)?

DOM is a programming interface for web documents and it represent the structure of HTML document as like tree structure.

1.getElementById

4.querySelector
5.querySelectorAll

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique by which attach a single event listener to a parent element instead of multiple child Elements.
Event delegation is use for:
1.Reduce number of event listener.
2.Handling similar elements.
3.Handling click on list.
